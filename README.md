Super Market Sales Dashboard

This project is an interactive sales dashboard designed to provide comprehensive insights into supermarket sales performance. It helps visualize key metrics like total sales, profit, product performance, sales types, and payment modes.

Features
Total Sales: $401,411.90
Total Profit: $68,907.90
Profit Margin: 21%
Monthly Sales & Profit Trends
Daily Sales Patterns
Top Products and Categories
Sales Type Distribution (Direct Sales, Online, Wholesaler)
Payment Mode Split (Cash vs Online)
Category Revenue Breakdown
Technologies Used
Power BI (or specify your visualization tool)
Excel / CSV data processing
Data visualization and analysis techniques
How to Use
Load your sales data (format compatible with the dashboard).
Select the year, month, sales type, and payment mode filters to customize your view.
Analyze key insights to drive business decisions.
Purpose

This dashboard aims to help retail managers and analysts monitor performance, identify trends, and optimize sales strategies using clear visual analytics.
